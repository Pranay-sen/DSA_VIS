# Code Visualizer

A Java Swing application that visualizes code execution similar to Python Tutor (pythontutor.com). This application allows users to input Java or Python code and visualize its execution step by step, showing memory models, variable states, and object references.

## Features

- Support for Java and Python code visualization
- Step-by-step code execution
- Visual representation of memory model
- Display of frames, objects, and references
- Code editor with syntax highlighting
- Execution controls (First, Prev, Next, Last)

## Requirements

- Java 8 or higher
- For Python code execution: Python 3.x installed and available in PATH

## How to Run

```
java -jar CodeVisualizer.jar
```

Or run the `CodeVisualizer` class directly from your IDE.
